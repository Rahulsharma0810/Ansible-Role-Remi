Ansible-Role-Remi
=========
Role to install and setup remi repo in Debian and Rpms, Remi enable wide variety package along with native pacakge manager.

Example Playbook
----------------

    - hosts: YOUR-HOST-OR-GROUP
      roles:
    	- Ansible-Role-Remi
